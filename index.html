What is dpip.lol ?

DPIP.lol (Accronym for DouxxPI Private domains) is a general purposes domain used by douxxpi.
This website, and mostly all subdomains.dpip.lol are hosted on a raspberry pi 5, so no 100% uptime is guaranteed.

API

dpip.lol is based on the 2peek.me API to get a fast, reliable and secure DNS managment API.
However, this API is private and you cannot generate an API key.
